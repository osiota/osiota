# Meta data

This is a minimal set of metadata.

## Global types

* data = time series data (plotable)
* info = static data
* status
* state
* object
* area
* trigger
* rpc

* log

### data (time series)

* energy.data = Energy consumption
* temperature.data = Temperature
* airflow
* light

* simple = just the state


### info (more or less static)
